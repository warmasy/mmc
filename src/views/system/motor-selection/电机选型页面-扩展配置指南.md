# 电机选型页面 - 功能扩展与配置指南

> 本文档说明如何新增选型模型、修改参数、调整计算逻辑，以及各组件的职责关系。

---

## 一、目录结构总览

```
motor-selection/
├── index.vue                          # 入口页面：模型选择器 + 公共信息栏
├── models/
│   ├── index.js                       # 模型注册中心（所有选型模型在此注册）
│   ├── servo-motor/                   # 伺服电机选型（示例）
│   │   ├── config.js                  # 参数配置（输入/输出/默认值/注释）
│   │   ├── calculator.js              # 纯计算逻辑（不依赖 Vue）
│   │   └── index.vue                  # 业务逻辑层（组装配置 + 计算 + 通用布局）
│   ├── stepper-motor/                 # 步进电机选型（框架）
│   └── linear-motor/                  # 直线电机选型（框架）
└── components/
    ├── MotorCalcLayout.vue            # 通用布局组件（左右双卡片 + 按钮 + 状态栏）
    ├── ParamInputPanel.vue            # 参数输入面板（从 JSON 动态渲染输入控件）
    └── ResultOutputPanel.vue          # 结果展示面板（从 JSON 动态渲染结果）
```

---

## 二、核心设计思想

**配置驱动渲染**：所有界面元素（输入框、结果展示、注释）都从 JSON 对象自动生成，
新增参数**不需要修改模板代码**，只需改 `config.js` 中的配置对象。

**组件完全复用**：三个模型（伺服/步进/直线）共享同一套 `MotorCalcLayout` + `ParamInputPanel` + `ResultOutputPanel`，
各自只写**配置 + 计算函数**。

### 步骤 2：创建 config.js —— 定义参数配置

```js
// ==================== 减速机选型 - 参数配置 ====================

// 输入参数配置（左侧卡片）
export const inputParams = {
  "计算模型": {
    type: 'image',
    items: {
      "计算模型": { src: '/path/to/gearbox-model.png', desc: '减速机模型示意图' }
    }
  },
  "运动参数": {
    type: 'number',
    items: {
      "输入转速n1": { key: 'inputSpeed', value: 1500, unit: 'r/min', step: 1, min: 0, desc: '输入轴转速' },
      "输出转速n2": { key: 'outputSpeed', value: 150, unit: 'r/min', step: 1, min: 0, desc: '输出轴转速' },
      "传动比i": { key: 'ratio', value: 10, unit: '', step: 0.1, min: 1, desc: '减速比' }
    }
  },
  "负载参数": {
    type: 'number',
    items: {
      "输出扭矩T2": { key: 'outputTorque', value: 100, unit: 'N·m', step: 0.1, min: 0, desc: '输出轴扭矩' },
      "安全系数Sf": { key: 'safetyFactor', value: 1.5, unit: '', step: 0.1, min: 1, desc: '安全系数' }
    }
  }
}

// 计算结果配置（右侧卡片）
export const outputParams = {
  "运动参数": {
    type: 'readonly',
    items: {
      "输入转速n1": { key: 'inputSpeed', unit: 'r/min', desc: '输入轴转速' },
      "输出转速n2": { key: 'outputSpeed', unit: 'r/min', desc: '输出轴转速' },
      "传动比i": { key: 'ratio', unit: '', desc: '减速比' }
    }
  },
  "选型结果": {
    type: 'result',
    items: {
      "输入扭矩T1": { key: 'inputTorque', value: '0.000', unit: 'N·m', desc: '输入轴所需扭矩' },
      "额定功率P": { key: 'ratedPower', value: '0.000', unit: 'kW', desc: '额定功率' }
    }
  }
}

// 底部注释（右侧卡片底部）
export const resultNotes = [
  '计算结果仅供学习参考，请勿用于其它用途'
]

// 从 inputParams 自动提取默认值生成 form 对象
export function getDefaultForm() {
  const form = { projectName: '', projectNo: '' }
  for (const group of Object.values(inputParams)) {
    if (group.type === 'number') {
      for (const item of Object.values(group.items)) {
        if (item.key) {
          form[item.key] = item.value !== undefined ? item.value : ''
        }
      }
    }
  }
  return form
}

// 从 outputParams 自动提取默认值生成 result 对象
export function getDefaultResult() {
  const result = {}
  for (const group of Object.values(outputParams)) {
    if (group.type === 'result') {
      for (const item of Object.values(group.items)) {
        if (item.key) {
          result[item.key] = item.value !== undefined ? item.value : ''
        }
      }
    }
  }
  return result
}
```

**参数字段说明**：

| 字段 | 必填 | 说明 |
|------|------|------|
| `key` | ✅ | 绑定到 `form` / `result` 对象的字段名，英文，唯一 |
| `value` | ✅ | 默认值（数字输入为数值，结果展示为字符串如 `'0.000'`） |
| `label` | ✅ | 显示名称，如 `"行程L"`，可包含符号下标 |
| `unit` | 可选 | 单位，如 `"mm"`、`"N·m"`，无单位填 `''` |
| `step` | 可选 | 步进值，如 `0.1`、`1`，决定鼠标滚轮调整幅度 |
| `min` | 可选 | 最小值，如 `0`，配合 `max` 限制输入范围 |
| `max` | 可选 | 最大值 |
| `precision` | 可选 | 小数位数，如 `4` 表示保留 4 位小数 |
| `desc` | 可选 | 鼠标悬停提示文字，如 `"行程长度"` |
| `showIf` | 可选 | 条件显示，值为 `form` 中某个字段的值，如 `showIf: 'screw'` |
| `type` | 可选 | 特殊类型：`select`（下拉选择）、`radio`（单选） |
| `readonly` | 可选 | `true` 表示只读计算值（如惯量比） |
| `options` | 可选 | 下拉/单选的选项字典，如 `{ ac: '交流伺服', dc: '直流伺服' }` |

**分组 type 说明**：

| 分组 type | 用途 | 示例 |
|-----------|------|------|
| `image` | 图片展示 | 计算模型示意图 |
| `radio` | 单选按钮组 | 驱动方式选择 |
| `number` | 数字输入框 | 运动参数、负载参数 |
| `readonly` | 只读回显（输出） | 参数回显 |
| `result` | 计算结果（红色粗体） | 选型结果 |

### 步骤 3：创建 calculator.js —— 纯计算逻辑

```js
// ==================== 减速机选型 - 计算逻辑 ====================

/**
 * 减速机选型计算
 * @param {Object} form - 表单参数对象
 * @returns {Object} { result: 计算结果, status: 状态文本 }
 */
export function calculateGearbox(form) {
  const inputSpeed = form.inputSpeed || 0
  const outputSpeed = form.outputSpeed || 0
  const outputTorque = form.outputTorque || 0
  const ratio = form.ratio || 1
  const safety = form.safetyFactor || 1.5

  // 计算输入扭矩（考虑效率 η = 0.95）
  const efficiency = 0.95
  const inputTorque = (outputTorque * ratio / efficiency) * safety

  // 计算额定功率 P = T × n / 9550
  const ratedPower = (inputTorque * inputSpeed) / 9550

  const result = {
    inputTorque: inputTorque.toFixed(3),
    ratedPower: ratedPower.toFixed(3)
  }

  // 生成状态提示
  let status = '计算完成'
  if (inputTorque > 1000) {
    status += ' | 输入扭矩较大，建议选用大功率减速机'
  }

  return { result, status }
}

/**
 * 参数验证
 * @param {Object} form - 表单参数
 * @returns {string|null} 错误信息，null 表示验证通过
 */
export function validateParams(form) {
  if (form.inputSpeed <= 0) return '输入转速必须大于 0'
  if (form.outputSpeed <= 0) return '输出转速必须大于 0'
  if (form.outputTorque <= 0) return '输出扭矩必须大于 0'
  if (form.ratio < 1) return '传动比必须大于等于 1'
  return null
}
```

**计算函数规范**：
- 输入：接收 `form` 对象（所有参数值）
- 输出：返回 `{ result: {...}, status: '状态文本' }`
- `result` 对象的 `key` 必须与 `outputParams` 中 `result` 分组的 `key` 对应
- `status` 用于显示在左侧卡片 header 的状态提示区域

### 步骤 4：创建 index.vue —— 业务逻辑层

```vue
<template>
  <MotorCalcLayout
    :inputParams="inputParams"
    :outputParams="outputParams"
    :form="form"
    :result="result"
    :statusText="statusText"
    :resultNotes="resultNotes"
    @calculate="handleCalculate"
    @reset="handleReset"
    @help="showHelp"
  />
</template>

<script setup name="GearboxSelection">
import { ElMessage, ElMessageBox } from 'element-plus'
import MotorCalcLayout from '../../components/MotorCalcLayout.vue'
import {
  inputParams,
  outputParams,
  resultNotes,
  getDefaultForm,
  getDefaultResult
} from './config'
import {
  calculateGearbox,
  validateParams
} from './calculator'

const props = defineProps({
  projectName: { type: String, default: '' },
  projectNo: { type: String, default: '' }
})

const emit = defineEmits(['update:projectName', 'update:projectNo', 'refreshUUID'])

// 表单数据（响应式）
const form = reactive(getDefaultForm())
watch(() => form.projectName, (val) => emit('update:projectName', val))
watch(() => form.projectNo, (val) => emit('update:projectNo', val))

// 计算结果（响应式）
const result = reactive(getDefaultResult())

// 状态文本
const statusText = ref('等待输入参数...')
let isResetting = false

// 监听参数变化，提示重新计算
watch(form, () => {
  if (isResetting) return
  const current = statusText.value
  if (current !== '参数已更新，请重新计算' && current !== '已重置' && current !== '等待输入参数...') {
    statusText.value = '参数已更新，请重新计算'
  }
}, { deep: true })

// 计算函数
function handleCalculate() {
  emit('refreshUUID')  // 通知父组件刷新 UUID
  const error = validateParams(form)
  if (error) {
    statusText.value = '参数错误：' + error
    ElMessage.warning(error)
    return
  }
  try {
    const { result: calcResult, status } = calculateGearbox(form)
    Object.assign(result, calcResult)
    statusText.value = status
    ElMessage.success('计算完成')
  } catch (e) {
    statusText.value = '计算错误'
    ElMessage.error('计算错误: ' + e.message)
  }
}

// 重置函数
function handleReset() {
  isResetting = true
  Object.assign(form, getDefaultForm())
  Object.assign(result, getDefaultResult())
  statusText.value = '已重置'
  ElMessage.info('已重置')
  isResetting = false
}

// 帮助函数
function showHelp() {
  ElMessageBox.alert(
    `减速机选型帮助：\n\n1. 输入输入转速和输出转速\n2. 填写输出扭矩和安全系数\n3. 点击"计算"获取选型结果\n4. 根据结果查阅减速机手册确认`,
    '减速机选型帮助',
    { confirmButtonText: '知道了' }
  )
}

// 暴露给父组件（可选）
defineExpose({
  calculate: handleCalculate,
  reset: handleReset,
  help: showHelp,
  getStatus: () => statusText.value
})
</script>
```

**注意**：`import MotorCalcLayout from '../../components/MotorCalcLayout.vue'`
路径是 `../../`（向上两级到 `motor-selection/` 目录），不是 `../`。

## 四、修改现有模型的参数

### 4.1 添加一个新参数

编辑 `models/servo-motor/config.js`，在 `inputParams` 的某个分组中新增一行：

```js
"运动参数": {
  type: 'number',
  items: {
    // ... 已有参数
    "丝杠长度l": { 
      key: 'screwLength',      // 字段名（英文，唯一）
      value: 1000,             // 默认值
      unit: 'mm',              // 单位
      step: 1,                 // 步进值
      min: 0,                  // 最小值
      showIf: 'screw',         // 只在丝杠模式显示（可选）
      desc: '丝杠总长度'        // 鼠标悬停提示（可选）
    }
  }
}
```

**自动生效**：
- `getDefaultForm()` 会自动提取 `value` 作为默认值
- `ParamInputPanel.vue` 会自动渲染新输入框
- 无需修改 `index.vue` 模板

### 4.2 修改参数默认值

直接改 `value`：

```js
"行程L": { key: 'stroke', value: 500, ... }  // 500 → 800
```

### 4.3 修改参数单位

```js
"最高速度Vmax": { key: 'maxSpeed', value: 0, unit: 'm/s', ... }  // mm/s → m/s
```

### 4.4 删除一个参数

直接从 `items` 中删除该行，`getDefaultForm()` 会自动排除。

### 4.5 添加条件显示参数

使用 `showIf` 字段：

```js
"丝杠导程Pb": { key: 'screwLead', value: 10, unit: 'mm', showIf: 'screw', ... }
```

当 `form.motionMode !== 'screw'` 时，该参数自动隐藏。

## 七、常见问题

### Q1：新增参数后界面没有显示？

检查 `config.js` 中的 `getDefaultForm()` 是否正确遍历了该参数所在的分组。
只有 `type: 'number'` 和 `type: 'radio'` 分组会被自动提取默认值。

### Q2：计算结果没有显示？

检查 `calculator.js` 返回的 `result` 对象的 `key` 是否与 `outputParams` 中定义的 `key` 完全一致（大小写敏感）。

### Q3：状态提示没有更新？

确保 `index.vue` 中 `watch(form, ...)` 监听器存在，且 `isResetting` 标志正确管理。

### Q4：条件显示参数（showIf）不生效？

`showIf` 的值必须与 `form` 中对应字段的值严格相等（如 `showIf: 'screw'` 对应 `form.motionMode === 'screw'`）。

### Q5：路径报错（Failed to resolve import）？

`models/xxx/index.vue` 中导入通用组件的路径必须是 `../../components/xxx.vue`（向上两级），不是 `../components/xxx.vue`。

### Q6：如何调整输入框宽度？

修改 `ParamInputPanel.vue` 中 `el-input-number` 的 `style="width: 110px"`，或给 `config.js` 中的参数添加 `width` 字段（需同步修改组件支持）。

### Q7：如何添加新的分组 type？

如需添加新的渲染类型（如 `slider` 滑块），需修改：
1. `ParamInputPanel.vue` 或 `ResultOutputPanel.vue` 添加新的 `v-else-if` 分支
2. `config.js` 中使用新的 `type`
